JWT & OAuth2 Security

표준 보안 인증 아키텍처 구축

서버 확장을 고려한 Stateless 인증과 소셜 로그인 통합으로 보안성을 강화했습니다.

  • Token Architecture: Access / Refresh Token을 활용한 세션리스 환경.
  • OAuth2 Integration: 카카오/구글 소셜 로그인 연동 및 인증 정보 관리.
  • Security Filter: 요청 단위의 토큰 검증 필터 체인 자동화.
protected void doFilterInternal(...) { String token = resolveToken(request); if (token != null && jwtProvider.validateToken(token)) { var auth = jwtProvider.getAuthentication(token); SecurityContextHolder.getContext().setAuthentication(auth); } }
